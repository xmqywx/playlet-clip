# Playlet-Clip 商业化交付手册

## 定位

Playlet-Clip 不是单纯卖源码，而是面向短剧 CPS、书单号、图文号和矩阵号团队的“自动解说剪辑交付工具”。客户购买的结果是更快产出可测试素材，减少人工听写、写解说、配音、剪片的时间。

## 30 天收入路径

### 优先客户

1. 短剧 CPS 操盘手：已有投流或自然流账号，缺稳定素材产能。
2. MCN/工作室：有剪辑团队，但希望把粗剪、解说、配音自动化。
3. 个人副业号主：愿意为部署、调参、模板和首批样片付费。

### 可售套餐

| 套餐 | 价格建议 | 交付内容 | 适合客户 |
| --- | ---: | --- | --- |
| 本地部署包 | 999-1999 元 | Mac/Windows/云服务器部署、豆包 API 配置、Edge-TTS、基础使用培训 | 个人号主 |
| 样片启动包 | 1999-3999 元 | 部署 + 3 套解说风格 + 10 条样片生产流程 | 小团队 |
| 工作室自动化包 | 5000-15000 元 | 私有部署 + 风格模板 + 批量处理脚本 + 1 周调优 | MCN/矩阵团队 |
| 月度维护 | 500-2000 元/月 | API/模型/提示词/报错维护，新增风格 | 已成交客户 |

## 演示流程

### 演示前准备

1. 准备一段 30-90 秒短剧素材，放到 `data/input/`。
2. 启动 Web UI：`uv run python -m playlet_clip.main`。
3. 打开 `http://localhost:7860`，确认设置页已有豆包 API、CPU ASR、Edge-TTS。
4. 准备三种风格：吐槽、悬疑、温情。

### 演示话术

核心话术：

> 这个工具把短剧二创的四个环节串起来：自动识别字幕、自动写解说、自动配音、自动合成成片。你不用先换剪辑团队，可以先用它跑 10 条样片，看哪种文案风格和节奏更容易出数据。

对个人号主：

> 你买的不是软件，是一套能在你电脑上跑的素材生产流程。我帮你部署好，再帮你调 3 个风格模板，你每天只需要换素材、点生成、挑能发的版本。

对工作室：

> 现在先不碰发布链路，只解决素材产能。等风格和留存数据稳定，再接你们的批量导入、账号矩阵和 Frida 发布自动化。

如果你现在有短剧素材，我可以免费用 1 条样片演示效果。满意再聊部署或代剪。

### 生成今日外联批次

用高优先级客户清单生成 CSV 跟进表、今日批次说明和逐客户私信：

```bash
uv run python scripts/build_outreach_batch.py \
  --prospects examples/high-priority-prospects.json \
  --output-dir ../../output/playlet-clip-outreach-2026-04-24 \
  --batch-name "Playlet-Clip 第一批高意向触达" \
  --free-sample-slots 3 \
  --package-path ../../output/playlet-clip-demo/customer-package
```

生成后优先使用：

- `outreach-tracker.csv`：跟进状态表。
- `today-batch.md`：当天触达顺序和执行规则。
- `messages/`：每个渠道一条可复制私信。

现在开放 3 个测试名额：适合短剧 CPS、书单号、矩阵号团队。可以先发 1 条素材，我做样片看效果。

## 成交判断

优先推进满足这些条件的客户：

- 已经在做短剧/CPS/矩阵号，而不是只想了解 AI。
- 手上有素材，能马上发 1 条测试视频。
- 愿意为部署、调参或样片付费，不要求免费定制全流程。
- 能接受先人工下载/上传，后续再自动发布。

暂缓客户：

- 只问“能不能保证爆单”。
- 要求绕过平台风控或批量违规搬运。
- 没素材、没账号、没预算，只想学习。
